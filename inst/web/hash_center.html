Returns the center of a geohash

Usage

hash_center(x)


hc_cmp_mem(x)

Arguments

x
a string representing a geohash

Description

Given a geohash this function will calculate the center lat/lon coordinate. To get the bounds and the error, use the function dcd(x).

Use this version with repeated calls to hash_center when you might be decoding the same hash more than once.
